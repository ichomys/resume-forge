A — Amber Reference Modified

Accent: Amber #E8952A Columns: 27% / 73% Spacing: Standard Header: Geometric accent shape

Faithful to your reference design with the column ratio narrowed. Left sidebar is lean — just enough for contact, skills, and section labels. Right column maximizes space for achievements and experience. This is the default.

B — Slate Professional

Accent: Slate Blue #3B5F8A Columns: 27% / 73% Spacing: Standard Layout: Identical to A

Same layout as Direction A with the slate-blue color scheme. Reads more corporate and technical — well suited for engineering, enterprise, financial, or government roles where calm professionalism is the signal.

C — Forest Trust

Accent: Forest Green #2E6B47 Columns: 27% / 73% Spacing: Standard Layout: Identical to A

Forest green accent — grounded, trustworthy. Reads well for roles in sustainability, operations, civic tech, or any context where reliability matters more than energy. A distinct alternative that doesn't shout.

D — Charcoal Minimal

Accent: Charcoal #3D3D3D Columns: 27% / 73% Spacing: Standard Effect: All-neutral palette

No color accent — pure black/gray. Maximum professionalism, zero visual distraction. Ideal for conservative industries (legal, finance, government) or any context where the work should be the only thing that stands out.

E — Amber Compact

Accent: Amber #E8952A Columns: 27% / 73% Spacing: Tight (10px section gap) Font: 9px body

Same amber design with reduced spacing and slightly smaller type. Fits more content per page — useful for roles requiring a longer experience section or when generated content is dense. Good "overflow" option for content-heavy runs.

F — Amber Bold Header

Accent: Amber #E8952A Columns: 27% / 73% Header: Full-width amber band Name: White on amber

Full-width amber header band with white name text. More assertive and visually distinctive. Body layout identical to Direction A. Best when you want the name to be unmissable at first glance — strong for creative, product, or startup roles.
